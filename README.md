# SFML-Chess-Game
Chess Game in C++ using SFML

This is a console + SFML based Chess game written in C++. It allows two players to play a full game of chess with:
* Legal move validation
* Check, checkmate, and stalemate detection
* Piece capturing and scoring
* Pawn promotion
* Highlighted possible moves

How to Run
* Make sure you have SFML library installed on your system.
* Open Chess.cpp in your IDE (like Visual Studio, Code::Blocks, or others).
* Make sure the images folder (piece textures) and the .ttf font file are in the same folder as Chess.cpp.
* Compile and run the program

Controls
* Click on a piece to select it.
* Click on a highlighted square to move the piece.
* Highlights:
* Green box: legal move
* Red box: captures an opponent piece

Notes
* Ensure SFML is correctly linked to run the graphical interface.
* Scores are calculated when pieces are captured.
